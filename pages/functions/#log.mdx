# #log

#log will log a message to the console, similar to print() in python.

## Usage
```jacob copy
#log[message;uppercase\lowercase;color]
```

## Parameters
Field | Type | Description | Required
--- | --- | --- | --- |
message | string | The message that will be sent to the console. | yes 
uppercase\lowercase | string | "lowercase"\"uppercase". Sends to the console as the option. | no
color | string | "blue", "red" or "yellow" is currently supported. | no

## Example(s)
This will log the text "Hello World!" uppercased and in blue:
```jacob copy
#log[Hello World!;uppercase;blue]
```
